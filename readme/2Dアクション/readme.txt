現在制作中の作品です。
現在はα版くらいの段階です

操作方法
移動：A→左に移動
	  D→右に移動
ジャンプ：W
ゴール(仮)に入る：S
弾の切り替え：F1
弾はノーマルと拡散弾、必殺技にする予定のホーミング弾です。
切り替え順
ノーマル→ホーミング弾→拡散弾→ノーマル…

参考にしている作品は「スーパーペーパーマリオ」と「カンパニーマン 社畜の下剋上」です。
見てほしいところはホーミング弾にベジェ曲線を入れているところや
ステージをCSVファイルを読み込んでマップを作っているところを見てほしいです。

テクスチャを同じものを使っていたりなどの見づらいところが多々ありますがご了承頂きたく存じます。
遊び要素が少ないですが、そこは順次いれていきたいと考えております。
